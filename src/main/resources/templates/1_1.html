将 t_user 用户信息到 MySQL 中，其他三个文件及，t_click，t_loan 和 t_loan_sum 导入到 HDFS 中。

1.启动Mysql服务

/home/aura/script/start_mysql.sh

2.使⽤mysql链接

mysql -hhadoop01 -uhadoop -phadoop

3.连接库

use group3;

4.创建表

CREATE TABLE t_user(
uid int(11) NOT NULL DEFAULT '0',
age int(2) DEFAULT NULL,
sex int(2) DEFAULT NULL,
active_date date DEFAULT NULL,
limit_amount double DEFAULT NULL,
PRIMARY KEY (uid)
)

5.把csv文件导入到mysql

LOAD DATA LOCAL INFILE
'/home/aura/group3/data/jdd_dataset/t_user.csv'
INTO TABLE group3.t_user
FIELDS TERMINATED BY ','
ENCLOSED BY '"'  LINES
TERMINATED BY '\n';

6.把其余数据放到HDFS上

hdfs dfs -put jdd_dataset /group3
hdfs dfs -put t_click.csv /group3/warehouse
hdfs dfs -put t_loan.csv /group3/warehouse
hdfs dfs -put t_loan_sum.csv /group3/warehouse
